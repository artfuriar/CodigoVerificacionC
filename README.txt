PROGRAMA QUE PIDE UN CÓDIGO Y COMPRUEBA SI ES CORRECTO#

Instrucciones:
Compila el programa. Puedes hacerlo, por ejemplo desde la terminal, con sudo gcc Codigo_Verificacion_C.c -o nombre_que_quieras_darle_sin_extension
Para ejecutar, desde el directorio en el que se encuentra el programa: ./Codigo
Si quieres ejecutarlo desde cualquier directorio añade su ruta a la variable PATH (recuerda que esta variable vuelve a sus valores por defecto  al cerrar la sesión, por lo que deberás
añadir su ruta en .bashrc para que el cambio sea permanente para tu usuario).
